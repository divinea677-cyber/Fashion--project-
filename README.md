# Fashion-Deepedia 👗
Developer:-Divine  
Location:** University of Uyo (UNIUYO)
#Project Overview
Fashion-Deepedia is a multi-page web application that explores the history, culture, and modern trends of fashion. This project focuses on a mobile-first user experience and clean UI design.
#Features
* Interactive Navigation:** Seamlessly switch between the Home page, Trends, and Author profile.
*Cultural Focus:Detailed research on West African textiles like Ankara and Kente.
*Contact Integration:** Functional WhatsApp Business and Instagram connectivity.
Smart UI-Floating "Back to Top" button for easy scrolling.
Tech Stack
HTML
 CSS3 (Flexbox & Fixed Positioning)
